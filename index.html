Features

왜 FestMap인가요

🗺️
지도 중심 UI
앱을 켜면 지도가 전부입니다. 불필요한 리스트, 탭바 없이 축제 마커만 눈에 들어옵니다.
📍
실시간 축제 마커
한국관광공사 TourAPI 기반으로 지금 이 순간 전국에서 열리는 축제를 실시간으로 표시합니다.
📸
풍부한 축제 정보
마커 탭 한 번으로 이미지 갤러리, 날짜, 장소, 요금, 주최자 정보를 한눈에 확인하세요.
🍜
근처 맛집 추천
축제 주변 5km 이내 음식점을 자동으로 추천해드립니다. 축제 전후 식사까지 완벽하게.
🔍
기간별 필터
오늘, 이번 주, 이번 달 단위로 보고 싶은 축제만 골라보세요. 계획에 딱 맞는 축제를 찾아드립니다.
🎯
MZ 감성 디자인
감각적인 UI와 부드러운 애니메이션으로 축제 정보를 탐색하는 경험 자체가 즐거워집니다.

How it works

딱 3초면 됩니다

01
앱 실행
켜는 순간 지도와 전국 축제 마커가 바로 표시됩니다.
02
마커 탭
가고 싶은 축제 마커를 탭하세요. 상세 정보가 아래에서 올라옵니다.
03
정보 확인
이미지, 날짜, 요금, 근처 맛집까지 한번에 확인합니다.
04
바로 출발
길찾기 버튼으로 카카오맵과 연동해 바로 이동하세요.
